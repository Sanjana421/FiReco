# FiReco AI Assistant

FiReco is an AI-powered financial product recommendation platform that recommends personalized credit cards using semantic search, embeddings, explainable AI, and LLM-generated summaries.

## Features

- Semantic recommendation engine using Sentence Transformers
- Personalized credit card recommendations
- Explainable AI recommendation reasoning
- FastAPI backend API
- Streamlit interactive frontend
- SQLite product storage
- LLM-generated recommendation summaries

## Tech Stack

- Python
- FastAPI
- Streamlit
- Sentence Transformers
- Scikit-learn
- SQLite
- OpenAI API

## Architecture

User Query
↓
Embedding-based Semantic Search
↓
Recommendation Ranking
↓
Explainability Layer
↓
LLM Summary Generation
↓
Frontend Display
